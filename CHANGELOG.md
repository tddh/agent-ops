# Changelog

## [0.1.0] — Unreleased

### Added
- 35 MCP tools for terminal session management (host, session, pane, window, file transfer)
- TLS + QUIC dual-channel transport (terminal over TCP/TLS, files over QUIC/UDP)
- SQLite-based audit logging with CLI query/stats/cleanup
- Cross-compilation support for Linux x86_64 (musl)
- systemd service deployment with auto-certificate generation
- JWT (HS256) and static token authentication
- Host registry with group/tag/label filtering
